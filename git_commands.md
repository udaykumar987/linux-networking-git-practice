# Git Commands

git init
Purpose: Initialize repository.

git clone
Purpose: Download repository.

git status
Purpose: Check repository state.

git add
Purpose: Stage changes.

git commit
Purpose: Save changes.

git log
Purpose: View history.

git branch
Purpose: Manage branches.

git switch
Purpose: Change branches.

git merge
Purpose: Merge branches.

git push
Purpose: Upload changes.

git pull
Purpose: Download and merge changes.

git fetch
Purpose: Download changes without merging.
