# Linux Commands

## System Information

whoami
Purpose: Shows current user.

hostname
Purpose: Displays server hostname.

uname -a
Purpose: Displays OS and kernel information.

## Disk Monitoring

df -h
Purpose: Shows filesystem usage.

du -sh *
Purpose: Shows directory sizes.

## Memory Monitoring

free -h
Purpose: Displays memory usage.

## Process Monitoring

ps -ef
Purpose: Lists running processes.

top
Purpose: Real-time monitoring.

## File Search

find /home -name "*.log"
Purpose: Finds log files.

grep "error" app.log
Purpose: Searches text inside files.

## Log Monitoring

tail -f /var/log/syslog
Purpose: Real-time log monitoring.
