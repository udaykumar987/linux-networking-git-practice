# Networking Commands

ip a
Purpose: Shows network interfaces.

hostname -I
Purpose: Displays server IP.

ping google.com
Purpose: Checks connectivity.

ip route
Purpose: Displays routing table.

nslookup google.com
Purpose: DNS lookup.

ss -tulnp
Purpose: Shows open ports.

curl -I https://google.com
Purpose: Checks HTTP response.

traceroute google.com
Purpose: Traces network path.
